# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Components
- `BookCard`: Reusable component to display individual book information

## Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

```
/src

  /data

  index.css

package.json
```

## Notes
- Ensure `node` and `npm` are installed.
- Before submission compress the project folder into `booklibrary_<yourname>.zip` and **remove the `node_modules` folder** from the zip.
